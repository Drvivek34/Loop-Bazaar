A repeatable agent workflow that debugs Git hooks configuration in Spring Boot and stops when target threshold is reached.

## 🏁 Implementation Steps
1. Scan the Spring Boot codebase for active Git hooks configuration.

4. Rerun the verification command to check the updated state.
5. Log output details and repeat until no deviations remain.
