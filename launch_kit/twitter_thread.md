### Tweet 2 (The Core Concept)
What is an agent loop?
Instead of a chatbot generating a single response, an agent runs in a Thought-Action-Observation cycle.

Loop-Bazaar converts raw prompts into structured loops featuring:
1. Use Cases
2. System Prompts
3. Checklist steps
4. Strict exit criteria 🛑

---

### Tweet 3 (Examples)
Take "The Docs Sweep" loop:
Whenever documentation sweeps are needed, the agent reviews the codebase, updates stale READMEs/examples, verifies commands, and opens a reviewable PR.

The loop terminates ONLY when the PR is opened. Bounded, safe, and deterministic.
